# Research Workspace

This folder is the working area for ongoing research.

We will use it for:

- market research
- user research synthesis
- product hypothesis testing
- academic literature reviews
- competitor tracking
- trust and safety investigation
- Python scripts for analysis and data collection

## Current Starting Point

Our first structured research artifact lives here:

- [Foundational Research Artifact](research-artifact-2026-03-23.md)

## How We Will Use This Folder Later

Future research can include:

- `.md` files for briefs, notes, summaries, interview synthesis, and literature reviews
- `.py` files for scraping, analysis, scoring experiments, clustering, or visualization

## Suggested Naming Pattern

- `YYYY-MM-DD-topic.md`
- `YYYY-MM-DD-topic.py`

Examples:

- `2026-03-25-user-interview-notes.md`
- `2026-03-26-competitor-comparison.md`
- `2026-03-27-trust-signal-analysis.py`

## Suggested Structure

- `research/README.md` for the research index
- `research/notes/` for raw notes
- `research/reports/` for polished writeups
- `research/scripts/` for Python analysis code
- `research/data/` for research-only datasets if we need them later

## Rule

Every new research file should answer one of these:

- what do we know
- what do we believe
- what evidence supports it
- what still needs validation
